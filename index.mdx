## ‚ö° Quick Start

Get your first transaction running in 60 seconds:

<Steps>
  <Step>
    ### Get API Key
    
    Visit the [Mr. Monei Dashboard](https://monei.cc) to get your API key.
    
    <Warning>
      Never expose your API key in client-side code or commit it to version control.
    </Warning>
  </Step>
  
  <Step>
    ### Initialize SDK
    
    <CodeGroup>
      ```typescript
      import MoneiSDK from 'monei-sdk';
      
      const sdk = new MoneiSDK({
        apiKey: process.env.MONEI_API_KEY!
      });
      ```
    </CodeGroup>
  </Step>
  
  <Step>
    ### Make Your First Call
    
    <RequestExample>
      ```typescript
      async function main() {
        try {
          const user = await sdk.user.getCurrentUser();
          console.log(`Welcome ${user.data.firstName}!`);
          
          const wallet = await sdk.walletAccount.getWallet();
          console.log(`Naira Balance: ‚Ç¶${wallet.nairaBalance.toLocaleString()}`);
        } catch (error) {
          console.error('Error:', error);
        }
      }
      
      main();
      ```
    </RequestExample>
    
    <ResponseExample>
      ```json
      {
        "data": {
          "firstName": "John",
          "lastName": "Doe",
          "email": "john@example.com"
        }
      }
      ```
    </ResponseExample>
  </Step>
</Steps>

<Tip>
  **Try it yourself**: Copy the code above, add your API key, and run it with `ts-node` or `node`.
</Tip>

    ```typescript
      amount: 5000
    ```

    <ParamField body="amount" type="number" required>
      Amount in Nigerian Naira (‚Ç¶)
    </ParamField>

  <Accordion title="Peer Transfer">
    ```typescript

    await sdk.payout.peerTransfer({
      receiver: "user@example.com",
      amount: 1000,
      transactionPin: "123456"
    });
    ```
  </Accordion>
  
  <Accordion title="Bank Account Verification">

    ```typescript
      bank: "058" // GTBank code
    });
    
    console.log(`Account Name: ${account.data.accountName}`);
    ```
  </Accordion>
